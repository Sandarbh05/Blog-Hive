BlogHive

A clean and minimal blogging application built with React, Redux, React Router, TailwindCSS, and Appwrite (Auth + Database + Storage).
Users can create, edit, delete, and read blog posts with image support and a smooth, responsive UI.

Features

User authentication (Signup, Login, Logout)

Create, edit, delete blog posts

Upload and display featured images

View all posts & individual post pages

Simple and intuitive UI

Tech Stack

Frontend: React, Redux, React Router, TailwindCSS

Backend: Appwrite (Authentication, BaaS, Storage)

State Mgmt: Redux

Forms: React Hook Form

Project Purpose

A beginner-friendly full-stack blog platform showcasing clean architecture, component reusability, and integration with Appwrite as a backend-as-a-service.
